Bio

Julien Viet works at Red Hat and contributes to the open source Vert.x and Ceylon projects. Julien has been developing open source software since 2002, he contributed to various project such JBoss Application Server 3, JBoss Portal or GateIn. Julien holds an engineering degree from Institut National des Télécommunications.

Open Source Involvement

I like Open Source Software and am/was a contributor of several projects

  • Vert.x: a lightweight, high performance application platform for the JVM that's designed for modern mobile, web, and enterprise applications.
  • Ceylon: Ceylon is a language for writing large programs in teams.
  • CRaSH: The shell for the Java Platform.
  • GateIn: Technical lead working mainly on the portal, portlet container.
  • Reflext: A framework abstracting the various Java reflection API.
  • Wikbook: A set of tools for producing Docbook documentation from wiki files and Java source code.
  • JBoss Portal: Project founder and leader until August 2008.
  • JBoss Application Server: Contributions to the project around 2002, mainly JMX and JNDI.
  • Cargo: Minor patches (shared classpath feature).
  • Spaz: Minor contributions to Spaz a nice Twitter client (URL rewriting feature, marked-as-read feature, dock icon badging).
